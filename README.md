# zzz-Banyue-character-guide
本攻略旨在针对**般岳**角色优化官方的技能描述以及搓招思路。为防止您在浏览本攻略时看到一系列专有名词而感到杂乱，本攻略规定以下专有名词称呼：

> [!IMPORTANT]
> A：指的是点按**普通攻击**按键触发的[普通攻击：峥嵘]
> 
> A'：指的是点按**特殊技**按键触发的[普通攻击：崔巍]
> 
> a：指的是[普通攻击]
> 
> q：指的是特殊技
> 
> Q：指的是闪能足够时的特殊技，即强化特殊技
> 
> 嗔火：指的是可积累的点数
> 
> 怒相：指的是**嗔火**积累到120点数时触发的状态
> 
> 指令：指的是如aQa、QQa、QaQ、aaQ这样的触发相应攻击的指令

## 普通攻击
### [普通攻击：引火]
不接战状态时，生命大于50%且**嗔火**不足120点时，长按a发动：
1. 消耗最多50%生命值回复**嗔火**，达到120点时直接进入**怒相**
2. 进入战场时获得一次释放该招式的机会，也就是说整场战斗只能释放一次

### [普通攻击：倾山]
> 招式动作为轰出巨大的山体攻击敌人

**怒相**时，长按a消耗120点**嗔火**时发动，发动后退出**怒相**

### [普通攻击：摧岳]
> 招式动作为向前方打出珠子禁锢敌人

**怒相**时，a触发或通过指令的方式触发两次后，长按a发动，之后自动释放[普通攻击：倾山]

### [普通攻击：焚身]
不处于**怒相**且拥有120点**嗔火**时长按a发动，之后进入**怒相**，并对周身敌人造成伤害

## 冲刺攻击
### [冲刺攻击：飞砂]
闪避时点按a，或在任意强化特殊技发动时长按a发动，之后可以衔接A'的第2段攻击

## 闪避反击
### [闪避反击：扬砾]
触发时点按a，可直接衔接A的第3段攻击

## 强化特殊技
### [强化特殊技：地动]
> 招式动作为重击地面

闪能足够时：
1. 第3、4段A后点按Q发动（aaQ）
2. 闪避反击、快速支援后长按Q发动
3. 强化特殊技、普通攻击：焚身、连携技、大招、闪避反击、快速支援时，aQa发动
#### [强化特殊技：山摇·怒]
> 招式动作为音波

闪能足够时，发动[强化特殊技：地动]时，QQa发动

---
### [强化特殊技：山摇]
> 招式动作为掀出山体

闪能足够时：
1. 第1、2段A'后点按a发动（qqa）
2. 强化特殊技、普通攻击：焚身、连携技、大招、闪避反击、快速支援时，QQa发动
---
### [强化特殊技：论道]
> 招式动作为扔珠子

闪能足够时：
1. 第3、4段A'后长按a发动
2. 闪避反击快速支援后，长按a发动
3. 强化特殊技、普通攻击：焚身、连携技、大招、闪避反击、快速支援后QaQ发动
#### [强化特殊技：狮子吼·怒]

> 招式动作为音波

闪能足够时，发动[强化特殊技：论道]时，aaQ发动

---
### [强化特殊技：狮子吼]
> 招式动作为音波

闪能足够时：
1. 第1、2段A后点按Q发动（aaQ）
2. 强化特殊技、普通攻击：焚身、连携技、大招、闪避反击、快速支援时，aaQ发动
---
## 大招
若**嗔火**大于120点，发动大招后自动进入**怒相**

## 连携技
若**嗔火**大于120点，发动连携技后自动进入**怒相**

## 核心技
处于**怒相**时长按q，自动打出4种强化特殊技（地动、山摇、论道、狮子吼）和[普通攻击：倾山]

## 技能循环

### 整体循环
```bash
【开局准备】
进入战场后，优先长按a（引火）消耗生命回复嗔火来进入怒相。此招式整场战斗仅可使用一次。

【怒相中爆发】
进入怒相后，立即长按q触发自动连段：倾山 → 地动 → 山摇 → 论道 → 狮子吼 → 倾山。此连段结束时自动消耗120嗔火并退出怒相。

【积累与再入】
爆发结束后退出怒相，嗔火归零。此时需迅速积攒嗔火至120点以准备下一轮循环：
• 优先使用普攻（A/A'）穿插闪避反击，稳定获取嗔火
• 若闪能充足，优先释放任意强化特殊技（地动/山摇/论道/狮子吼）获取较高嗔火
• 若闪能不足，优先进行闪避循环/快速支援循环
• 若已拥有120嗔火但未处于怒相，可通过长按a（焚身）主动进入
• 积攒过程中，若嗔火已超过120点，可主动释放大招或连携技，释放后自动进入怒相

【循环】
嗔火满120后，返回【怒相中爆发】继续循环。
```

---

### 闪避 & 支援衔接
```bash
【闪避循环】
闪避 + 点按a（飞砂）→ 衔接A'第2段 → 点按q（山摇）→ 继续输出或进入怒相

【闪避反击循环】
触发闪避反击 + 点按a（扬砾）→ 衔接A第3段 → 点按Q（地动）→ 继续输出或进入怒相

【快速支援循环】
快速支援后 → 长按a/Q（分别对应论道、地动）或输入对应技能指令 → 触发强化特殊技 → 继续输出或进入怒相
```
---

### 其他关键点
- **大招/连携技**：若嗔火>120，释放后自动进入怒相 → 可立即接长按 q 爆发
- **指令派生灵活使用**：在强化特殊技、焚身、连携、大招、闪反、快速支援后，可用 **aQa / QQa / QaQ / aaQ** 快速接对应特殊技
